I loved Python ever since I was first introduced to it in my Intro to EECS class. We used it all the time at MIT: for research, for homework, for doing Matlab work without Matlab... Since then, I've used it as my main scripting language and in small projects. I was a Lab Assistant for that same EECS course and the one after it, 6.02. I also helped with an Introduction to Python class during our January period. More recently, I've been learning how to build and deploy Flask applications and experimenting with pandas and scikit-learn. In February 2017, I joined the InfoScout Insights team as a back-end engineer, which will involve working with Django, Celery, and other interesting tools.

I've worked with Java since 2011, ever since I took 6.005 (back then I used Eclipse and hated it). After college, I worked on the ADF Shared Components (ADFShare) team at Oracle, from July 2013 until January 2017. We used (and developed components for) JDeveloper. In fact, one of my first projects at Oracle was to add internal static-member audits to JDeveloper.

I have my qualms about Java, but Java 8 looks really promising, with type annotations, lambda expressions and method references.

I used to be anti-JS in my high-school days, but since then we've had jQuery, node, and ES2015. I've made my peace with it now, but I only use it when I need to.
